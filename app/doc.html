Welcome to the Sanskrit Bot API

Methods

Identify a Character

Endpoint: http://localhost:5000/identify

Request type: POST

Request body: JSON with field "character", e.g. 

  {
    "character": "अ"
  }

Response: JSON with character details, e.g. 

  {
    "Character Symbol": "अ",
    "Name": "a",
    "Type": "Vowel",
    "Pronunciation": "/a/ (like 'a' in 'about')"
  }

Exceptions: If the character is not recognized, the system returns a "Character not found" message.

Transliterate a Word

Endpoint: http://localhost:5000/transliterate

Request type: POST

Request body: JSON with field "word", e.g. 

  {
    "word": "नमस्ते"
  }

Response: JSON with transliterated word, e.g. 

  {
    "transliterated_word": "namaste"
  }

Exceptions: If the input contains non-Devanagari characters, the system returns an error message.
